# PostOffice

A CLI application for browsing and executing Postman collections with a k9s-like interface.

## Features

- Browse Postman collections and folders
- Execute HTTP requests
- Interactive TUI interface

## Installation

```bash
go build -o postOffice
```

## Usage

```bash
./postOffice
```

## Commands

- `:collections` - List available collections
- `:request` - List folders and requests
